There should be a feature that allows administrators or exam officers to set the number of questions each student can access per course. For example, a course may contain up to 1000 questions in the question bank, and the admin / lecturer will select 100 but each student will only be given access to 30 questions during the exam. this is so that questions can be suffled properly 
The system should randomly select 30 questions from the full pool of 100 questions for each student, while also shuffling the order of both the questions and answer options. This will ensure that every student receives a different combination of questions and options, rather than all students answering the same 30 questions with only the options rearranged.
